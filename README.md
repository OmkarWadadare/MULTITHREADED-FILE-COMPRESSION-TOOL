## Multithreaded File Compression Tool is a high-performance C++ application designed to compress and decompress files efficiently by leveraging multithreading.
The project focuses on improving execution speed by dividing large files into smaller chunks and processing them concurrently using multiple threads. This approach significantly reduces compression and decompression time compared to single-threaded implementations.
The application supports both compression and decompression operations and demonstrates measurable performance gains through time comparisons. It highlights the practical use of concurrency in system-level programming and introduces optimization techniques used in real-world utilities like file archivers and data transfer systems.

## Objectives
To understand multithreading concepts in C++
To implement parallel file processing using threads
To design and implement file compression and decompression logic
To measure and compare performance between single-threaded and multithreaded approaches
To optimize I/O operations for large files

## Features
Compress large files into smaller sizes
Decompress files back to original format
Multithreaded processing for faster execution
Chunk-based file splitting and parallel handling
Performance benchmarking (time measurement)
Command-line or menu-driven interface
Error handling for file operations and thread execution
